UI STANDARD

A high-performance, object-oriented UI framework for robust application development. Built-in callback frames, secure persistence, and zero-bloat architecture.

View Features 
-- Initialize the Spawn Framework
local Spawn = loadstring(game:HttpGet("https://spawn.framework/core.lua"))()

-- Create a Window
local Win = Spawn.Create("System Control")
local Tab = Win.Tab("Main Settings")

-- Add Components with One-Line Callbacks
Tab.Toggle("fly_mode", "Flight Systems", function(state) 
    print("Flight systems active: " .. tostring(state))
end)

Tab.Slider("speed", "Velocity", 16, 100, 50, function(val)
    print("Speed set to: " .. val)
end)

Callback Architecture

Don't manage event loops manually. The UIManager handles input states, focus, and clicks, calling your functions only when needed.

Industrial Design

Pre-styled with a deep void aesthetic, reinforced plating headers, and high-visibility text. Looks professional out of the box.

Auto-Persistence

State management is built-in. Toggles, sliders, and textboxes automatically save their state to JSON and restore upon reload.
